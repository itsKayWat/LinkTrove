QuickBookmark

QuickBookmark is a Python-based command-line tool designed to help you manage and access your bookmarks effortlessly. Whether you're bookmarking URLs, file paths, or scripts, QuickBookmark provides a simple and efficient way to keep your important resources organized and easily accessible.

Why QuickBookmark?

Managing bookmarks manually can become cumbersome, especially when dealing with multiple types of resources like websites, documents, and scripts. QuickBookmark simplifies this process by providing a unified interface to add, remove, list, and load your bookmarks directly from the terminal.

Features

  • Add Bookmarks: Easily add new bookmarks with a unique identifier.
  • Remove Bookmarks: Remove bookmarks that are no longer needed.
  • List Bookmarks: View all your saved bookmarks in an organized manner.
  • Load Bookmarks: Quickly access your bookmarks, whether they are URLs, file paths, or scripts.
  • Persistent Storage: Bookmarks are saved in a JSON file for persistent storage across sessions.

Installation

Clone the Repository:

git clone https://github.com/yourusername/QuickBookmark.git

Navigate to the Directory:

cd QuickBookmark

Install Required Packages:

pip install -r requirements.txt

Usage

Run the script using Python:

python bookmarks.py

Follow the on-screen menu to add, remove, list, or load bookmarks.

Menu Options

  • Add a Bookmark: Add a new bookmark by providing a unique number and the bookmark value (URL, file path, or script path).
  • Remove a Bookmark: Remove an existing bookmark by entering its number.
  • List Bookmarks: Display all saved bookmarks.
  • Load a Bookmark: Access a bookmark by its number. Depending on the type, it will open in your default browser, execute a script, or open a file.
  • Exit: Exit the QuickBookmark application.

Requirements

Ensure you have Python installed. Install the required packages using the following commands:

pip install -r requirements.txt

Example Use Case

Customer Situation:

Imagine you're a developer who frequently accesses various resources like project documentation (PDFs), internal websites, and Python scripts for automation. Instead of searching through folders or bookmarks in a browser, QuickBookmark allows you to access all these resources directly from the terminal with simple commands.

Steps:

  1. Add Bookmarks: 
    python bookmarks.py
# Choose option 1 to add a bookmark
# Enter a unique number, e.g., 1
# Enter the bookmark value, e.g., https://github.com
  2. List Bookmarks: 
    python bookmarks.py
# Choose option 3 to list bookmarks
  3. Load a Bookmark: 
    python bookmarks.py
# Choose option 4 and enter the bookmark number, e.g., 1
# The default browser will open GitHub
